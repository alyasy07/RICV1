# Image Placement Guide

## Required Images for Welcome Page

Place your images in the following locations within the `public/images/` directory:

### Header Logos (Top of page):
1. **logo-malaysia.png** - Malaysian government logo
2. **logo-umk.png** - Universiti Malaysia Kelantan logo
3. **logo-geric.png** - GERIC logo

### Background Image:
4. **building-placeholder.jpg** - Main background image of the building (the one shown in your screenshot)

### Footer Logos (Bottom of page):
5. **footer-logo-1.png** through **footer-logo-6.png** - Various organizational logos

## Instructions:

1. Create the `public/images/` directory if it doesn't exist
2. Save all your logo images and background image in that directory
3. Rename them according to the names listed above, or update the file paths in `welcome.blade.php`

## Temporary Placeholder:
The page currently uses a light blue background color as a placeholder for the building image.
Once you add your actual building image, it will display automatically.

## Image Specifications:
- **Header logos**: Recommended height 50px, PNG format with transparent background
- **Background image**: Minimum 1920x1080px, JPG format
- **Footer logos**: Recommended height 35px, PNG format with transparent background
